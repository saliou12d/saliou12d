## 📌 Descripción del Proyecto
**Zenthra.Core_Security** es una plataforma backend enfocada en **seguridad avanzada, correlación de eventos y analítica inteligente**.  
Desarrollada con **Python, FastAPI, SQLAlchemy** y **JWT**, sigue principios de **POO y arquitectura limpia** para soluciones escalables y seguras.
